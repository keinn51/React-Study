## React-Study

집현전 사람들을 위한 react study 자료. 친해지길 바라 프로젝트 참여를 위해 팀원들에게 React 를 알려드립니다!

### 폴더 구조

-   member_practice : 참여한 팀원들의 소스 코드
-   practice_code : 강의에 사용한 코드들, 일별로 나뉘어 있음
-   lecture : 강의록이 적혀 있음

### 공부 순서

총 8강으로 진행됩니다.

1. React 사용 이유 및 설치
2. HTML + CSS 무조건 알아야 하는 것!
3. 상태와 Effect
4. Axios로 서버 데이터 가져오기
5. recoil 사용해보기
6. 타입스크립트
7. 실전 코드 읽기
